\subsection{Detailed Description}
A\-P\-P\-L\-I\-C\-A\-T\-I\-O\-N\-: Gives structures and main functions to create random belief functions. This module provides with methods to randomly generate mass functions. It is roughly based on common algorithms to do so. An interesting paper on the subject is \char`\"{}\-Random Generation of Mass Functions\-:
